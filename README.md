## Description

A logger for usdt contract.

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Notes

This logger is only for ERC-20 events.

I didn't do this project only for logging but as a boilerplate for a blockchain application.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
